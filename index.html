Lyra Discord Bot

Lyra is a multifunctional Discord bot I developed with a focus on usability, modular design, and integration with a custom web interface. Built in Python using discord.py, it handles asynchronous events, modular command groups (cogs), and persistent server-specific configurations. On the frontend, I created a responsive interface with HTML, CSS, and JavaScript, connected to the bot through a REST API that allows real-time interaction from the browser. Users can view stats, run commands, or adjust settings without needing Discord. I also built a dynamic UI that preserves navigation state and syncs active panels. One of the main challenges was ensuring smooth, persistent synchronization between Discord and the web client. This project highlights my full-stack development skills, combining real-time interaction, async programming, and user-focused design. While the code is not yet public, the website will be live soon.

Stack Overflow Questions

This is a simple I built to fetch and serve recent Stack Overflow questions using the Stack Exchange public API. I developed it with Flask to create lightweight and fast endpoints that return JSON data containing titles, tags, links, and creation dates of the most recent questions. It is useful for integrating Stack Overflow updates into other platforms or tools, such as dashboards or bots. I kept the code modular and added error handling to ensure it remains reliable even if the external API fails or returns unexpected data.
